Usage Instructions:

1. Load or paste a JSON file with the required message structure.

2. To use the extension, highlight any text (minimum 3 letters) on a web page and press the "s" key.

3. A popup will appear with available messages filtered by your selection.

4. Click on a message to copy it to the clipboard.

JSON Format:

The JSON structure should be an object where:

- Each key is a category name (can be any name you choose)

- Each value is an array of string messages

        {
          "your-category-name": [
            "Message 1 for this category",
            "Message 2 for this category"
          ],
          "another-category": [
            "Message for another category"
          ]
        }
